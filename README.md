# Nano Coating for Glass | Benefits and Protection

Nano coating for glass creates a hydrophobic layer that repels water, dirt and stains.

## Benefits
- Easy to clean glass
- Water repellent surface
- Protection against stains
- Longer lifespan

## Where to use
- Windows
- Shower glass
- Glass railings
